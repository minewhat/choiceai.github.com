Introduction

ChoiceAI SDK

ChoiceAI SDK provides a way for developers to modify the default behavior of ChoiceAI on web pages.

Custom Javascript

window.scroll(0, 200)

Custom JavaScript allows you to do custom operations when automation events are fired.

To create a Custom JavaScript, goto automations page in choice ai app -> select an automation -> click custom js tab -> choose an action and code.

ChoiceAI Custom JavaScript condition allows you to write your own JavaScript that is evaluated for each experiment visitor trigger/action.

Actions available for custom javascript

  • onOpen
  • onClose
  • onInteraction

Utility Functions

CAISDK.isActive(experimentId)

Utility functions allows you to get/set info/state about currently available automations.

  • isActive
  • isShown
  • trigger
  • hide

Automation Sets

Automation sets allow you to target your experiment for a uniquely defined set of pages or visitors. Please check your plan type to learn whether this feature is available.

Automation sets allows you to make sure that only one automation under the set is shown to the user at a time.

Automation sets can be of two types

  • Rules
  • Custom Code

In rules method, you can define required priority on the automations. The system will automatically pick the highest available priority automation.

In custom code method, you will get an interface to write your own custom logic. In this method, you will have to define a function, which accepts two parameters -> params & callback.
